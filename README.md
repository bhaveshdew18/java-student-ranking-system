# 🎓 Student Ranking System (Java)

A console-based **Student Ranking System** developed in Java to demonstrate **Object-Oriented Programming (OOP)**, **Collections Framework**, **Comparable**, **Comparator**, **Exception Handling**, and **CRUD operations**.

This project allows users to manage student records and sort them using different criteria.

---

## ✨ Features

- ➕ Add a new student
- 📋 Display all students
- 🔍 Search student by Roll Number
- 📝 Update student marks
- ❌ Delete a student
- 📊 Sort students by Marks (Natural Ordering)
- 🔤 Sort students by Name
- 🔢 Sort students by Roll Number
- ⚠️ Basic input validation and exception handling

---
